⚙️ Configure o Firebase
Para usar o sistema real, você precisa conectar ao seu projeto Firebase. É gratuito e leva 5 minutos.
Passo 1: Acesse console.firebase.google.com
Passo 2: Crie um projeto novo (ex: agendapro-minha-barbearia)
Passo 3: Vá em Authentication → Ative o provedor E-mail/Senha
Passo 4: Vá em Firestore → Crie um banco de dados
Passo 5: Vá em Configurações do projeto → Seus apps → Adicione um app Web
Passo 6: Copie as credenciais e cole abaixo

As credenciais ficam salvas no seu navegador (localStorage). Não são enviadas para terceiros.
